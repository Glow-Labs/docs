---
title: "Get User"
api: "GET https://game-server-dev.fly.dev/api/user/{userId}"
description: "This endpoint gets User by User Id"
---
### Parameters
<ParamField header="Authourization" type="string" required>
  This is the Authourization token of customer.
</ParamField>
<ParamField path="userId" type="string" required>
  This is the id of the user
</ParamField>

### Response

<ResponseField name="name" type="string">
  Name of the User
</ResponseField>

<ResponseField name="email" type="string">
  Email of the User
</ResponseField>

<ResponseField name="image" type="string">
  Image of the User
</ResponseField>

<ResponseField name="customerId" type="string">
  Id of Customer User is Linked
</ResponseField>

<ResponseField name="userId" type="string">
  Id of the User
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request \

--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>'
```
