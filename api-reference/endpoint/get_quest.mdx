---
title: "Get Quest"
api: "GET https://game-server-dev.fly.dev/api/v1/quests"
description: "This endpoint get Quest by Quest Id"
---

### Headers
<ParamField header="Authorization" type="string" required>
  This is the Authorization token of customer.
</ParamField>

### Query Parameters
<ParamField query="customerId" type="string">
  This is customer Id for which customer user is associated with
</ParamField>
<ParamField query="questId" type="string">
  This is the Id of the Quest
</ParamField>
