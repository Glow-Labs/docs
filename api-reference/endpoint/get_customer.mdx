---
title: "Get Customer"
api: "GET https://game-server-dev-dev.fly.dev/api/v1/customer"
description: "This endpoint return Customer Id by passing discord guild id"
---
### Parameters
<ParamField query="discordGuildId" type="string" required>
  This is the guild of the discord server bot is installed
</ParamField>

### Response

<ResponseField name="customerId" type="string">
  Id of the customer
</ResponseField>

<RequestExample>
```bash Example Request
curl --location --request \
GET 'https://game-server-dev-dev.fly.dev/api/v1/customer?discordGuildId=12170430311741521' \
--header 'Content-Type: application/json' \
```

</RequestExample>

<ResponseExample>

```json Response
{
  "customerId": "65bcd9ab9f54b8ec4c9316cf"
}
```

</ResponseExample>
